# Changing Perspectives
 
El juego trata de que un cientifico te secuestra y te implanta un chip que hace que cuando intentas "saltar"(haciendo referencia al la tecla espacio) tu vista cambia a de una camara a otra, y deberas de completar pruebas para poder salir vivo del lugar
El juego termina en el nivel 2, pues no se ha implementado una manera de abrir la puerta de salida.
